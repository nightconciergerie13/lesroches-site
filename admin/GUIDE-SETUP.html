🛠 Guide de configuration

Les Roches Retreat — Backend Supabase + EmailJS · Temps estimé : 30–45 minutes

1
Créer le compte Supabase Gratuit
Base de données · Prix dynamiques · Disponibilités · Réservations
1
Aller sur supabase.com → Start your project

Créer un compte (GitHub recommandé). Gratuit jusqu'à 500 Mo et 50 000 lignes.

2
Créer un nouveau projet

Nom : les-roches — Choisir la région la plus proche (US East ou EU West) — Créer un mot de passe fort et le noter.

3
Exécuter le script SQL de setup

Dans Supabase → menu gauche → SQL Editor → New query → copier-coller le contenu du fichier admin/supabase-setup.sql → cliquer Run.

✅ Ce script crée automatiquement les 4 tables : villas, blocked_dates, reservations, airbnb_ical — avec toutes les règles de sécurité.
4
Récupérer les clés API

Supabase → Project Settings (roue dentée en bas à gauche) → API — Copier :

  • Project URL → ressemble à https://abcdef.supabase.co
  • anon public (sous "Project API keys") → longue clé JWT
5
Coller les clés dans le fichier js/config.js

Ouvrir ce fichier dans un éditeur texte (TextEdit sur Mac, Notepad sur PC) et remplacer :

const SUPABASE_URL = 'https://TON-ID.supabase.co'; // ← coller ton Project URL
const SUPABASE_ANON = 'eyJhbGciOiJIUzI1...'; // ← coller ton anon public key
6
Créer ton compte admin (pour accéder au panneau admin)

Supabase → Authentication → Users → Invite user → entrer ton email ian.gay@perrimond.eu → tu recevras un email pour définir ton mot de passe.

⚠️ Ce compte est uniquement pour toi — les clients n'ont pas besoin de compte. Le site public fonctionne sans authentification.
2
Configurer EmailJS Gratuit · 200 emails/mois
Emails de confirmation automatiques aux clients et à toi
1
Créer un compte sur emailjs.com

Plan gratuit = 200 emails/mois, largement suffisant pour commencer.

2
Ajouter un service email

EmailJS → Email Services → Add New Service → choisir Gmail (ou autre) → connecter ton compte email → noter le Service ID généré.

3
Créer le template pour le client

EmailJS → Email Templates → Create New Template — Exemple de template :

Objet : Votre demande de réservation — Les Roches Retreat

Bonjour {{guest_name}},

Nous avons bien reçu votre demande de réservation pour la villa {{villa_name}}.

📅 Arrivée : {{checkin}}
📅 Départ : {{checkout}}
🌙 Durée : {{nights}}
👥 Voyageurs : {{guests}}
💰 Total : {{total}}
💳 Acompte (50%) : {{deposit}}

Nous vous contacterons sur WhatsApp pour confirmer la disponibilité et vous envoyer les coordonnées bancaires.

Virement à effectuer sous 48h :
Bénéficiaire : NIGHT CONCIERGERIE SRL
IBAN : CR19 0114 0000 7812 3915 72
SWIFT : BCCECRSJ
Banque : BANCO LAFISE

À très bientôt,
L'équipe Les Roches Retreat

Dans EmailJS, To Email : {{guest_email}} — From Name : Les Roches Retreat

Noter le Template ID de ce template (ex: template_abc123).

4
Créer le template pour toi (alerte propriétaire)

Créer un 2ème template "Nouvelle réservation" avec les mêmes variables mais envoyé à ian.gay@perrimond.eu. Noter son Template ID.

5
Récupérer ta Public Key

EmailJS → Account → API Keys → copier la Public Key.

6
Coller les IDs dans villas.html

Ouvrir villas.html et trouver ce bloc (vers la ligne 686) :

const EMAILJS_PUBLIC_KEY = 'ta_public_key';
const EMAILJS_SERVICE_ID = 'service_xxx';
const EMAILJS_TEMPLATE_GUEST = 'template_guest_xxx';
const EMAILJS_TEMPLATE_OWNER = 'template_owner_xxx';
3
Configurer la synchronisation Airbnb
Bloquer automatiquement les dates Airbnb sur ton site
1
Récupérer l'URL iCal de chaque villa sur Airbnb

Sur Airbnb → Tableau de bord hôte → Sélectionner une annonce → Calendrier → Disponibilité → Synchroniser les calendriers → Exporter le calendrier → copier l'URL iCal.

2
Coller les URLs dans le panneau admin

Ouvrir admin/index.html → se connecter → onglet Sync Airbnb → coller l'URL iCal pour chaque villa → cliquer Sync.

💡 À faire une fois par semaine pour garder les disponibilités à jour. Tu peux aussi cliquer Sync avant de confirmer une réservation cliente.
4
Tester le système complet
Vérifier que tout fonctionne avant de lancer
1
Tester le panneau admin

Ouvrir admin/index.html → se connecter avec ton email/mot de passe → vérifier que les 6 villas apparaissent dans l'onglet Tarifs.

2
Tester la réservation client

Ouvrir villas.html → cliquer "Book Now" → entrer des dates → vérifier que les prix s'affichent → cliquer "Reserve" sur une villa → remplir le formulaire → soumettre.

Vérifier ensuite dans le panneau admin (onglet Réservations) que la réservation est bien enregistrée.

3
Tester le changement de prix

Admin → Tarifs → modifier le prix d'une villa → sauvegarder → rafraîchir villas.html → le nouveau prix doit s'afficher immédiatement.

📋 Récapitulatif des fichiers modifiés

• js/config.js — Clés Supabase (à renseigner)

• villas.html — IDs EmailJS à renseigner + Supabase déjà intégré

• admin/index.html — Panneau admin complet (déjà lié à config.js)

• admin/supabase-setup.sql — Script à exécuter une seule fois dans Supabase
